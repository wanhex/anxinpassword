# 安心密码

## 功能描述

帮助用户记住各种网站、App等账户密码。

为了保证用户密码的安全性，在本地采用加密数据库，数据库密码采用一机一密随机高强度密码、结合使用系统KeyStore（RSA非对称加密）、锁屏验证等多重方案做保护。

用户本地记录的密码可以同步至个人百度云盘账户，并在必要时从百度云盘恢复。

## 开源代码贡献流程

任何软件开发者都可以发现并修改软件中的bug，新增feature，帮助安心密码提升用户密码的安全性。

贡献流程参考GitHub Flow：

1. 创建分支
2. 在新分支中进行开发新feature或bug修复
3. commit/push修改
4. 提交Pull Request
5. 项目管理员Review后合并入库
